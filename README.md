Personal Portfolio Website

A modern, fully responsive personal portfolio website built to showcase my career journey, technical skills, projects, and professional certifications as an Associate Software Engineer.

🚀 Key Features:

Smart Navigation: An intuitive, auto-hiding navigation bar that stays out of the way while scrolling down and reappears when scrolling up.
Smooth User Experience: Native smooth scrolling for all internal page navigation.
Interactive Design: Engaging UI with smooth scroll-based animations (Intersection Observer API) and parallax hover effects on project cards.
Integrated Contact Form: Built-in contact form powered by Formspree, allowing recruiters to reach out directly.
Responsive Layout: Professional layout with a modern dark theme using CSS Flexbox and Grid.
Quick Access: "Scroll to Top" button for better navigation on longer pages and easy access to my resume.

🛠️ Tech Stack:

Frontend: HTML5, CSS3, JavaScript
Deployment: GitHub Pages
